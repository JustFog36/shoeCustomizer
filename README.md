# shoeCustomizer
A website to change the background color, built using react

## Usage
1. CLone the Repo
2. run 'npm i' and then 'npm run build 'in the root directory
3. Npm start and then go to site on your local machine.

## Technologies Used
React, express, Mongo, Node, NPM
